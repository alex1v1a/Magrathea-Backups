## Company Overview

Vectarr is an online manufacturing platform connecting customers to machined parts production with instant quoting. The platform eliminates the need for 2D drawings for users with limited CAD skills while supporting both low-quantity prototypes and detailed bidding for larger production runs.

### Core Value Propositions

1. **Instant Quoting:** Upload CAD files and receive immediate quotes
2. **No 2D Drawings Required:** 3D models sufficient for quoting
3. **Low Quantity Friendly:** Prototypes and small runs supported
4. **Detailed Bidding Portal:** For larger quantities and assemblies
5. **Vetted Machine Shop Network:** Quality-assured manufacturing partners

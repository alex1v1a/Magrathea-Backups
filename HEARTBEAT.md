## Response Protocols

### If Machine Shop Replies
1. Read email carefully
2. Draft response using appropriate persona
3. Update Mission Control board
4. Update shop profile with new info
5. Notify alex1v1a if urgent or complex

### If Customer Inquiry Received
1. Identify correct alias (sales/info/support)
2. Draft response with proper signature
3. Create/update customer profile
4. Update Mission Control board
5. Log in daily notes

### If Technical Issue
1. Document issue details
2. Check if resolution is documented
3. Draft response or escalate
4. Update knowledge base with solution

## Quiet Hours

## Escalation Triggers

Escalate to alex1v1a immediately if:
- Legal/compliance question
- Data breach or security concern
- Customer dispute requiring human judgment
- Machine shop requests phone call
- Any request to circumvent platform terms

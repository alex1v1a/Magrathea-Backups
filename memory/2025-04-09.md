### How to Create New API Keys

When new API keys are needed, follow these steps:

1. **Navigate to OpenAI API Settings:**
   - Go to: https://platform.openai.com/api-keys

2. **Create New Key:**
   - Click the "Create new secret key" button
   - Give it a descriptive name (e.g., "Vectarr-Production", "Vectarr-Dev")
   - Select appropriate project if using projects
   - Click "Create secret key"

3. **Copy and Store Securely:**
   - **IMPORTANT:** The key is only shown once at creation
   - Copy it immediately
   - Store in a secure password manager or environment variable
   - Never commit API keys to version control

4. **Set Environment Variable:**
   ```bash
   # Windows (PowerShell)
   $env:OPENAI_API_KEY = "sk-..."
   
   # Windows (Command Prompt)
   set OPENAI_API_KEY=sk-...
   
   # Linux/macOS
   export OPENAI_API_KEY="sk-..."
   ```

5. **Test the Key:**
   ```bash
   curl https://api.openai.com/v1/models \
     -H "Authorization: Bearer $OPENAI_API_KEY"
   ```

### Security Best Practices

- Rotate keys periodically (every 90 days recommended)
- Use separate keys for different environments (dev/staging/prod)
- Never share keys in chat messages or emails
- Monitor usage in the OpenAI dashboard for unexpected activity
- Revoke keys immediately if compromised

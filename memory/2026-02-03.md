## Commands Reference

### Kanban
```bash
# Manual sync
node marvin-dash/scripts/kanban-sync.js

# Check WSL status
powershell -File wsl-keepalive.ps1 -Check
```

### WSL Keep-Alive
```powershell
# Install scheduled task
powershell -File wsl-keepalive.ps1 -Install

# Check status
powershell -File wsl-keepalive.ps1 -Check

# Run service manually
powershell -File wsl-keepalive.ps1 -Service
```

### Cron Management
```bash
# List cron jobs
openclaw cron list

# Run kanban sync now
openclaw cron run kanban-sync

# Disable/Enable
openclaw cron disable kanban-sync
openclaw cron enable kanban-sync
```

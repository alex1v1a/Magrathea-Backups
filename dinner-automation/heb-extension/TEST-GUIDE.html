🛒 HEB Auto-Cart v2.0 - Test Guide

The extension has been completely rebuilt with item-by-item verification!

Setup Steps

Step 1: Reload Extension

  1. Go to chrome://extensions/
  2. Find "HEB Auto-Cart"
  3. Click the 🔄 reload icon

Step 2: Sign In to HEB

  1. Go to heb.com
  2. Click "Sign In" in top right
  3. Enter your HEB credentials
  4. Verify you see your name in the top right corner

Step 3: Test with Sample Items

Copy this test data:

{ "items": [ {"name": "Milk", "searchTerm": "whole milk", "amount": "1 gallon"}, {"name": "Eggs", "searchTerm": "large eggs", "amount": "1 dozen"}, {"name": "Bread", "searchTerm": "white bread", "amount": "1 loaf"} ] }

Step 4: Run Extension

  1. Click the 🛒 HEB Auto-Cart icon in Chrome toolbar
  2. Paste the test data in the text box
  3. Click "Load Items"
  4. Click "Start Adding Items"
  5. Watch the console (F12 → Console) for detailed logs

What to Expect

  • ✅ Green outline on buttons when clicked
  • ✅ Console shows each step: "Searching...", "Clicking Add to Cart...", "Verified!"
  • ✅ Progress bar updates as items are added
  • ✅ Cart count increases as items are added

Troubleshooting

If nothing happens:

  1. Press F12 → Click Console tab
  2. Look for red error messages
  3. Check if you're logged in to HEB

If items search but don't add:

  1. Check console for "Add to Cart button" messages
  2. Look for green outlines on buttons (means click happened)
  3. Verify you're on the search results page

To debug in console:

Paste this and press Enter:
